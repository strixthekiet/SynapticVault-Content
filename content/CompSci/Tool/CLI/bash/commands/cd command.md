# cd command
### Description:
- Change the current working directory to directory. 
	- If directory is not supplied, the value of the `HOME` shell variable is used.
- 

### Synopis:
- `cd`
	- `[-L`
		- `|[-P`
		- `[-e]]`
	- `[-@]`
	- `[directory]`
		- Change the current working directory to directory.
		- If directory is not supplied, the value of the `HOME` shell variable is used
