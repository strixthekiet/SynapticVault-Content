# kubectl logs
### Description:
- Print the logs for a container in a pod or specified resource. 
### Synopsis:
- `kubectl logs`
	- `[-f]`
	- `[-p]
	- `(`
		- `POD `
		- `| TYPE/NAME`
	- `)`
	- `[-c CONTAINER]`: only optional if the pod has a single container
