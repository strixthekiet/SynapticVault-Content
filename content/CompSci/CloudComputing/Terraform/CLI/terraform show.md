# terraform show
### Description:
- Provide output from a state or plan file. 
- This can be used to inspect a plan to ensure that the planned operations are expected, or to inspect the current state as Terraform sees it.
### Synopsis:
- `terraform show`
	- `[options]`
	- `[file]`
### Options:
- `--json`
